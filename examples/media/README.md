# SafeOps Media Package

Use this folder to store non-sensitive demo media such as screenshots, short GIFs, or public demo video references.

Recommended files:

- `safeops-demo-video-link.txt`
- `incident-cockpit-screenshot.png`
- `github-actions-ci-screenshot.png`
- `investor-demo-terminal-screenshot.png`

Do not commit private customer logs, secrets, credentials, tokens, or production data.
