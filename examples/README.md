# SafeOps Examples

This directory contains static examples and generated-demo reference materials.

## Contents

| Path | Purpose |
|---|---|
| `audit-events/` | Example audit event JSON. |
| `incident-reports/` | Example incident report JSON. |
| `cicd/` | Example CI/CD deployment context. |
| `slack/` | Example Slack-style approval card. |
| `dashboard/` | Example Incident Cockpit HTML artifact. |
| `investor/` | Notes for the investor demo package. |

The live demo writes runtime output under:

```text
/tmp/safeops-demo
```

The investor demo package is generated under:

```text
/tmp/safeops-demo/investor-demo-package
```
